# Roadmap

## v1

- CLI-first passive OSINT orchestration.
- Local SQLite case storage.
- HTML, CSV, and JSON reports.
- Safe subprocess collector wrappers.
- Offline unit tests with fixture output.

## v1.x

- Broader parser compatibility across collector versions.
- Case listing and filtering in the CLI.
- Analyst annotations and severity overrides.
- Stronger report theming and screenshots.

## v2

- Experimental read-only API expansion.
- Optional scan-triggering API only with enforced authorization and passive mode.
- Passive SpiderFoot integration.
- Optional secure local-lab mode for controlled secret handling.
